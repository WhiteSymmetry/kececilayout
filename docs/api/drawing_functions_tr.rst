------------------------
``draw_kececi``
------------------------

.. autofunction:: kececilayout.draw_kececi

Grafi, belirli bir stille çizer. Otomatik olarak `to_networkx` kullanır.

Desteklenen stiller:

- ``'curved'``: Eğri kenarlar.
- ``'transparent'``: Uzaklığa göre şeffaflık.
- ``'3d'``: 3D heliks yerleşimi.
